================================================================================
ENCRYPTION DETAILS
================================================================================

LOCAL DATA ENCRYPTION:
  Algorithm:   Fernet (AES-128 CBC + HMAC-SHA256)
  Key Size:    256 bits
  KDF:         Scrypt
  
  Scrypt Parameters:
    - Salt:    16 bytes (128 bits) random
    - N:       2^14 = 16,384 iterations (brute-force resistant)
    - R:       8 (memory cost)
    - P:       1 (parallelization)
  
  Encrypted Files:
    - userinfo.enc:  User registration (name, ID, dept, email)
    - passcode.enc:  Current 8-character passcode
    - lockout.bin:   30-minute lockout timestamp
  
  NOT Encrypted (Audit Trail):
    - usb_security.log:  Activity log (must remain readable)

CREDENTIAL MANAGER:
  Encryption:  DPAPI (Windows system-level)
  Access:      Only authenticated user
  Storage:     HKCU\Credentials (binary encrypted blob)
  
  Stored Credentials:
    - master_password:   Used to encrypt local files
    - email_address:     SMTP sender email
    - email_password:    SMTP authentication password

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "Administrator privileges required"
  SOLUTION: Right-click setup_secure.bat or run.bat
            Select "Run as Administrator"

PROBLEM: "Python not found"
  SOLUTION: Install Python 3.8+ from python.org
            Add to PATH during installation
            Verify: python --version in Command Prompt

PROBLEM: "Email not sending"
  SOLUTION: 1. Configure in Settings (⚙️)
            2. For Gmail, use App Password (not regular password)
            3. Verify internet connection
            4. Check firewall allows port 587 outbound

PROBLEM: "USB won't toggle"
  SOLUTION: 1. Verify running as Administrator
            2. Check Registry: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\USBSTOR
            3. Restart computer to apply changes
            4. Check logs for errors

PROBLEM: "System locked"
  SOLUTION: Wait 30 minutes for lockout to expire
            OR delete: %APPDATA%\USBControlPro\lockout.bin
            NOTE: 3 more wrong attempts will lock again
