TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

PROBLEM: Windows says "This app couldn't start properly"
────────────────────────────────────────────────────────────────────────────
✓ Solution:
  1. Right-click usb_security_pro.exe
  2. Click Properties
  3. Click Unblock
  4. Try again as Administrator


PROBLEM: "Access Denied" or "Permission Denied"
────────────────────────────────────────────────────────────────────────────
✓ Solution:
  1. Must run as Administrator
  2. Right-click → "Run as Administrator"
  3. Normal user execution won't work


PROBLEM: GUI doesn't appear / stuck on startup
────────────────────────────────────────────────────────────────────────────
✓ Solution:
  1. Check AppData folder is writable
  2. Verify: C:\Users\[User]\AppData\Roaming\USBControlPro\ exists
  3. Check logs: %APPDATA%\USBControlPro\usb_security.log
  4. Reinstall if corrupted


PROBLEM: Email configuration not working
────────────────────────────────────────────────────────────────────────────
✓ Solution:
  1. Gmail: Enable "Less secure app access"
  2. Try: Generate App Password (Gmail security)
  3. Verify: SMTP server is smtp.gmail.com:587
  4. Enable: TLS encryption (automatic)
  5. Check: Firewall allows port 587 outbound


PROBLEM: Webcam not detecting for intruder photo
────────────────────────────────────────────────────────────────────────────
✓ Solution:
  1. Check if camera is connected
  2. Verify in Device Manager
  3. Grant Windows permissions to app
  4. App works without camera (just no intruder detection)
